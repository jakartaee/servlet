Jakarta EE Servlet Technology Compatibility Kit, Version 6.2
Release Notes, May 2024

Kit Contents

The Jakarta EE Servlet, Version 6.2 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE Servlet TCK tests signature, API, and End-to-End tests:
    • Signature test that checks that all of the public APIs are supported in the Jakarta EE Servlet Version 6.2 implementation that is being tested
    • API tests for the public APIs under the jakarta.servlet, jakarta.servlet.annotation, jakarta.servlet.descriptor and jakarta.servlet.http packages
    • Specification tests for testable assertions required by the Jakarta EE Servlet 6.2 Specification

Jakarta EE Servlet TCK Facts

The test suite bundle contains the following:

  • 1721 total tests (1720 tests and 1 signature test)

Platform Notes

The Jakarta EE Servlet TCK tests have been built with JDK 17 and tested with JavaTM Platform, Standard Edition 17 and 21.

The Jakarta EE Servlet TCK tests have been run on the following platforms:

  • Ubuntu Linux 22.04

The Jakarta EE Servlet TCK tests have been run against the following Jakarta EE Servlet 6.2 Compatible Implementations:

  • Apache Tomcat 11

Installing, Setting Up, and Running the JavaTM API for Servlet TCK

Refer to the Jakarta EE Servlet Technology Compatibility Kit, Version 6.2 User's Guide (or PDF) for complete instructions on installing, setting up, and running the Jakarta EE Servlet TCK.
